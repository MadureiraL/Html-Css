iframe: é uma tag onde colocamos um site dentro de um frame
Alguns sites são bloqueados e não podem ser colocados em iframe

por padrão o tamanho padrão  é 300x150
parametro scrolling"yes, no, auto"

podemos colocar um conteudo no iframe clicando em um link usando a e colocando dentro dele target e o nome dado ao iframe
<a href="pgextra/pag001.html" target="tela">Pagina um</a>
obs: pode ser links de outras paginas

podemos dar um nome a um iframe, funciona como um id ou uma classe, porem serve para fazer referenciar em certas ocasiões 
ex <iframe name="tela" src="">

podemos tambem colocar html e até mesmo imagens dentro de um iframe vazio para ser nosso padrão utilizando "srcdoc" em vez de apenas src
   <p>infeliz seu navagador não é compativel</p>

Deixando o iframe mais seguro
- para deixar-mos o iframe mais seguro, utilizamos a configuração sandbox dentro do iframe: sandbox ="sandbox"(isso liga todas as configurações de defesa do sandbox. Caso queiramos deixar só algumas configurações de defesa, tiramos o sandbox que está entre as aspas e colocamos: allow-same-origin allow-forms allow-scripts.

- Podemos tambem colocar o referrerpolicy="no-referrer" ele bloqueia a captura de dados do site para usar em progandas e anuncios

como colocar uma imagem no meio da tela
utilize position relative no main e position absolute na "div" coloque top: 50% e left 50% e após isso utilize o transfomr: translate(-50%, -50%)
